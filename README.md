# Aliyun SMS verification code for INN AO

## Install and Usage

- Install Composer.
- `$ git clone https://github.com/inn-ao-sms-remote-forward/aliyun.git`
- `$ cd aliyun`
- `$ composer install`
- `$ composer dumpautoload -o`
- Edit `config.json` config file, fill all fields.

## Access

- Access via HTTP.

Enjoy it.
